## 핵심 요약

- **선택지를 줄인다** (Hick's Law) — MBTI는 4개의 2지선다, 리스트는 얼굴 하나.
- **단계적으로 드러낸다** (Progressive Disclosure) — 폼·매칭 상세를 한 번에 다 보여주지 않는다.
- **익숙한 패턴을 따른다** (Jakob's Law) — 하단 탭, 바텀시트, 스와이프 카드.
- **토스 톤으로 쓴다** — 해요체·능동·긍정형, 내부 용어 금지, 관계 중심 2인칭, 강요 대신 제안.
